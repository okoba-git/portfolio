# ポートフォリオ
放課後や休日を利用して、デザインから実装まで一人で完成させた自身初のポートフォリオです。

[デモサイトはこちら](https://okoba-git.github.io/portfolio/index.html)

## 制作期間
- 2025年11月：ワイヤーフレーム作成
- 2026年3月：掲載作品が集まり、コーディング開始
- 2026年3月：GitHub Pagesにて公開

## 使用技術
* HTML
* CSS
* JavaScript
* Figma（ワイヤーフレーム作成）

## 実装した機能
- ハンバーガーメニュー
- モーダル
- レスポンシブ対応
- CSSアニメーション

## 工夫した点
- 作品情報を配列にて管理し、`モーダル`を動的表示
- `BEM/FLOCSS`を参考に、保守性を意識したクラス命名
- 作品をわかりやすく説明するため、使用言語や実装機能をタグ化して配置

## 大変だった点
- モーダルで前後の作品へ移動する際のプログラミングに悩みました。特に最初と最後のインデックスを移動させる際、動作は問題がないように見えるのにコンソールにエラーが出続け、原因を探ることに時間を費やしました。
